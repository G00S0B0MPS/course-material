# Assignment

:::TASK
Write a function `double(x)` that, given a number `x`, returns double this value.

Note that multiplication is represented by the symbol `*` in Python (as it is in most other programming languages.)
:::

:::USAGE

```python
>>> double(0)
0

>>> double(1)
2

>>> double(2)
4

>>> double(1111)
2222
```

:::
