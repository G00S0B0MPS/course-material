# Assignment

:::TASK
Write a function `triple(x)` that, given a number `x`, returns the triple of this value.

Note that  is represented by the symbol `*` in Python (as it is in most other programming languages.)
:::

:::USAGE

```python
>>> triple(0)
0

>>> triple(1)
3

>>> triple(2)
6

>>> triple(1111)
3333
```

:::
